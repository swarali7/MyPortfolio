Curriculum

Software Engineering

This course is focused on the principles of software development, including the full software development lifecycle, Agile methodologies, and managing large-scale software projects. It emphasizes problem-solving, design, testing, and ensuring high-quality code and documentation. Students will work on real-world projects and learn about various development tools, version control systems, and collaboration techniques.

Tech Stack & Skills: Git, GitHub, Agile/Scrum, JIRA, Unit Testing, Version Control, Software Design Patterns, Software Development Life Cycle (SDLC).

Web Application Development

This course teaches the key concepts of developing scalable and secure web applications. Topics include both front-end and back-end technologies, focusing on modern frameworks like React for the front-end and Node.js and Express.js for the back-end. Students will build full-stack applications and learn about client-server communication, database management, and web security practices.

Tech Stack & Skills: HTML, CSS, JavaScript, React.js, JavaScript, TypeScript Node.js, Express.js, REST APIs, MySQL.

Project-Based Learning

This is a hands-on course, where we applied software engineering knowledge to real-world projects. Working in teams, we used the Agile methodology to develop full-stack applications, from design and development to testing and deployment. Emphasis is placed on collaboration, communication, and using industry tools and practices to build production-level software.

Tech Stack & Skills: Full-Stack Development, React, Typescript, PostgreSQL, Web Security (OAuth2, JWT), Git, Agile/Scrum, CI/CD, JIRA, DevOps, Cloud Platforms (AWS/Heroku), Project Management, Collaboration Tools.
