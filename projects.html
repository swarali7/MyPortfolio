My Projects

Image Steganography Android Application

This project involves the development of an Image Steganography android application where users can hide secret messages within images and retrieve them later. The tool allows embedding text into an image using LSB (Least Significant Bit) encoding, which modifies the least significant bits of pixel color values to store hidden messages. The project utilizes Java for image manipulation and text encoding, and also features an intuitive GUI built with Android Studio to make it user-friendly.

Tech Stack: Java, Android Studio, MySQL Database

Banking Process Automation

This project involved automating repetitive banking processes, which reduced operational inefficiencies at Bank of America. I developed an automation system using Python, Oracle databases, and Quartz Scheduler to manage data workflows and streamline reporting. The system improved the overall speed of banking operations and decreased manual errors by automating tasks such as data entry and report generation.

Tech Stack: Python, Oracle Database, Quartz Scheduler, SQL
